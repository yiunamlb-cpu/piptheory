# playbook/

Macro frameworks distilled from primary sources. This is the agents' epistemic foundation — what they reason *with*, not just data they see.

Planned sections (each a separate file as it's written):

- `regime_identification.md` — Dalio quadrants, growth/inflation matrix, liquidity regimes
- `reflexivity.md` — Soros framework: how price action becomes fundamentals
- `asymmetry_hunting.md` — Druckenmiller pattern: when to concentrate, when to skip
- `cycles.md` — Marks: where in the cycle, second-level thinking
- `central_bank_reading.md` — how to parse FOMC/ECB/BoJ/BoE statements, dot plots, language drift
- `positioning.md` — COT extremes, dealer gamma interpretation, crowded-trade unwind dynamics
- `cross_asset_confirmation.md` — when DXY/yields/equities/gold tell consistent vs. conflicting stories
- `macro_history.md` — analog episodes (1998, 2008, 2015, 2020, 2022) and what they teach

This is the highest-value bespoke asset in the project. OSS gives us none of it.
