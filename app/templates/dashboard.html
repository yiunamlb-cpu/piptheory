    How to read this page
    

      
Focus list vs context. The system splits instruments
      into two groups:

      
    
        
  • Focus list ({{ active_universe|join(', ') }}) —
          the instruments you trade on FTMO. These get the full analysis:
          macro view, council debate (Bull / Bear / Judge), and a structural
          review of the chart that produces a Tradable / Watch / Skip verdict.
    • 
        
  • Context (everything else) — macro analysis only.
          No chart-level check. Used to keep the wider macro picture in view
          but not as actionable setups.
    • 
      

      
To change the focus list, edit
      ACTIVE_UNIVERSE in src/orchestration/pipeline.py.
      Currently kept small (3 instruments) while we calibrate output quality;
      can be expanded later.


      
Conviction is a 1-10 score. When the council debate
      ran (focus-list instruments), conviction is the Judge's final
      number after weighing bull and bear cases. Otherwise it's the
      Strategist's initial macro read. Same number wherever it appears on
      this page.


      
Run-to-run variance. Two consecutive pipeline runs
      can produce convictions that differ by 1 point even with the same data —
      this is normal LLM noise. A 6 vs 7 doesn't mean anything actionable
      changed. Look for movement of 2+ points or category changes
      (Watch ↔ Skip).


      
What's shown vs hidden. Visible above:

      
    
        
  • Tradable now — chart and macro both green-lit (focus list only)
    • 
        
  • Watch — bias supported, location not yet ready (focus list only)
    • 
        
  • Context with conviction ≥ 6 — strong macro view, no structural check
    • 
      

      
Collapsed below:

      
    
        
  • Skip — chart's against it — focus list, macro says yes but chart says no
    • 
        
  • Focus · no filter run — focus list with low macro conviction (no structural review needed)
    • 
        
  • Context with conviction < 6

Active positions

        

          Trades you've opened on FTMO. The system never executes — you record
          entries here so the system can flag when the macro thesis or chart
          structure has materially changed. Close marks a trade exit; closed
          trades go to the journal for monthly review.
        

      


      
      

        How to read this tab
        

          
What you see per position. Each position renders as
            a card with: a P&L row, a coloured pill at the top-right showing the
            recommended action, a one-sentence reason, and a metadata strip with
            the macro state, chart verdict, stop distance, and thesis at open.


          
What the action pills mean. The system runs a rule
            ladder against each open position on every page-load. Top of the ladder
            wins. Pills, in order of urgency:


          
    
            
  • 
              Emergency close
              Your structural stop has been breached. Don't deliberate —
              close per the pre-defined plan. Whole point of having a structural stop is
              you don't get to vote when price is at the level.
            
    • 
            
  • 
              Close
              Either macro view has flipped to the opposite direction at 6+/10
              conviction, OR the chart structure has eroded (Filter verdict turned to
              pass_despite_bias). The system's own analysis no longer supports
              the position. Exit; re-enter on a cleaner setup if your view holds.
            
    • 
            
  • 
              Trim
              Conviction has dropped 3+ points since open, or position is in
              meaningful profit with conviction slipping. Take some risk off; keep a
              residual on for the case where the thesis re-firms.
            
    • 
            
  • 
              Trail stop
              In profit (≥5%) with the macro thesis still intact. Consider raising
              the emergency stop to lock in gains. Don't tighten so much that normal
              volatility takes you out — the stop is structural, not technical.
            
    • 
            
  • 
              Review
              Soft alert. Macro view still aligned with the
              position, but the Filter says watch rather than
              tradable_now — chart has lost its clean location. No
              action required. Glance, note, move on. Only acts as a real
              signal if it persists for several days while P&L deteriorates.
            
    • 
            
  • 
              Hold
              Default. Thesis intact, no action needed. Most days will be Hold.
            
    • 
          


          
Reading the wobble. The macro conviction number on
            each card moves day-to-day. A 6→5→6→7 oscillation is normal LLM noise,
            not a real change in view — the system has hysteresis built in (Trim
            triggers on a 3+ drop from open, Close triggers on a 6+ opposite-direction
            reading). Don't react to single-point moves. Trade the band, not the digit.


          
What "macro now" tells you. The metadata strip on
            each card shows whether the latest run still aligns with your position
            direction, the current conviction, and how many points it's moved since
            you opened. Aligned means same direction; OPPOSED means
            the system's bias has flipped to the opposite of your trade.


          
Buttons on each card.

          
    
            
  • Mark closed — when you've actually exited on FTMO.
              Records exit price + reason; appends to the journal.
    • 
            
  • Add note — timestamped log entry. Use for
              "trimmed half here", "moved stop", "ignored advisor and was right",
              etc. This becomes your trade narrative for review.
    • 
            
  • Move stop / Set stop — update the structural
              invalidation level. The system never moves it for you.
    • 
            
  • Delete — only for mistakes (opened in error). If
              you actually closed the trade, use Mark closed so it goes to the journal.
    •
